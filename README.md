# Bank Transaction Analyzer

Простая и эффективная утилита на Python для обработки банковских операций. Проект демонстрирует навыки работы с алгоритмами, внешними API, различными форматами данных и автоматизированным тестированием.

## Технологии
* **Язык:** Python 3.12+
* **Управление зависимостями:** Poetry
* **Тестирование:** Pytest (физические файлы + моки внешних API)
* **Линтеры:** Flake8, Black, Isort
* **Библиотеки:** Requests (API), Pandas (для работы с XLSX/CSV)

## Основные возможности
* **Универсальный парсинг:** Чтение данных из форматов JSON, CSV и XLSX.
* **Безопасность:** Маскирование номеров карт (вид `XXXX XX** **** XXXX`) и счетов (`**XXXX`) для защиты персональных данных.
* **Фильтрация и поиск:** Гибкая сортировка операций по дате, статусу (`EXECUTED`, `CANCELED`) и текстовый поиск по описанию.
* **Конвертация валют:** Интеграция с внешним API для автоматического пересчета транзакций в рубли (RUB) по актуальному курсу.
* **Логирование:** Процесс обработки данных логируется (декораторы) для отслеживания ошибок и хода выполнения.

## Установка и запуск

1. Клонируйте репозиторий:
   ```git clone https://github.com/AJLbN0H/bank-transaction-analyzer.git```
   
3. Установите зависимости через Poetry:
   ```poetry install```

## Тестирование
Проект полностью покрыт Unit-тестами. Для проверки корректности работы и покрытия выполните:
```pytest --cov=src```
Отчет о покрытии доступен в формате HTML (директория htmlcov).
